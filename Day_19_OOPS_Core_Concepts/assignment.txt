Classwork Assignment

Question 1: Encapsulation Practice
Create a class called Student.

Requirements:
Create a private attribute called __marks.
Create a method set_marks() to update marks.
Create a method get_marks() to display marks.
Marks should only be accepted if they are between 0 and 100.
Display an appropriate message for invalid marks.

Question 2: Bank Account System
Create a class called BankAccount.

Requirements:
Create a private attribute called __balance.
Create the following methods:
deposit()
withdraw()
get_balance()
Deposit amount must be greater than zero.
Withdrawal amount cannot exceed the available balance.
Display an appropriate message when funds are insufficient.
